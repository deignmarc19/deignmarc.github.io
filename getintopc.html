GetIntoPC Redesign

A complete redesign of a software download website using Figma

About This Project

I chose GetIntoPC.com as an example of bad website design. The original site had too many ads, cluttered layout, poor typography and no visual hierarchy.

What I Improved

  • Clean modern navbar
  • Added search bar
  • Clear hero section
  • Organized software cards
  • Trust/safety banner
  • Better colors and fonts

Tools Used

  • Figma — for the redesign
  • HTML & CSS — for coding
